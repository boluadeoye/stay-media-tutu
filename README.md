# TÚTÙ | Premium Canned Cocktails
## Technical Assessment: Headless WordPress + Next.js 15

This project is a high-fidelity, performance-optimized landing page developed for the **STAY MEDIA** technical assessment. It demonstrates a modern decoupled architecture, prioritizing design precision and technical scalability.

### 🏗 Architecture Overview
- **Frontend:** Next.js 15 (App Router) utilizing React Server Components for optimized data fetching.
- **Backend:** Headless WordPress (REST API) used as a centralized data engine for content management.
- **Styling:** Tailwind CSS for utility-first, pixel-perfect responsive design.
- **Animations:** Framer Motion for cinematic, high-end editorial transitions.
- **Deployment:** Vercel Edge Network for global low-latency delivery.

### 💎 Key Technical Implementations
- **Absolute Figma Fidelity:** Replicated complex Bauhaus geometric patterns using custom-engineered SVG tiles and enforced strict editorial typographic ratios using the **Cormorant Garamond** and **Montserrat** font stacks.
- **Decoupled Data Flow:** Integrated **Advanced Custom Fields (ACF)** and **Custom Post Types (CPT)** to manage Services, Testimonials, and Hero content independently of the UI logic.
- **Performance Optimization:** Leveraged Next.js Image optimization and Server-Side Rendering (SSR) to ensure near-instant load times and high SEO scores.
- **Mobile-First Engineering:** Aggressive spatial compression and custom navigation logic to maintain luxury aesthetics across all viewports.

### 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure `NEXT_PUBLIC_WP_URL` in your environment variables.
4. Run development server: `npm run dev`.
